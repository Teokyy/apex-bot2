# 🤖 Telegram Авто-Постер & Бот Дней Рождения

Добро пожаловать! Данное приложение было успешно перенесено из фонового скрипта Python в полноценное интерактивное веб-приложение на базе **Full-Stack Node.js (Vite + React + Express)**.

Оно оснащено удобной веб-панелью управления, которая позволяет в реальном времени отслеживать статус бота, редактировать дни рождения персонажей, безопасно настраивать ключи доступа и просматривать логи.

---

## 🌟 Основные возможности

- ⚡ **Консоль управления в реальном времени**: Включайте/выключайте бота одной кнопкой, запускайте немедленную ручную проверку RSS-лент и отслеживайте время следующего запуска.
- 📺 **AI-Локализатор публикаций**: Интегрируется с **Google Gemini API** (через `gemini-2.5-flash`) или **Cerebras / другими OpenAI-совместимыми эндпоинтами** (`gemma-4-31b`) для генерации естественных обзоров аниме и игр на русском языке с использованием заданных промптов-персон.
- 🎮 **Интеграция с магазином Steam**: Автоматически ищет цены и ссылки на Steam для игровых публикаций.
- 🎂 **Поздравления аниме-персонажей с Днем Рождения**: Автоматически проверяет календарь и публикует красивые посты с поздравлениями в канал `@animuds`, когда наступает день рождения персонажа из базы данных.
- 📅 **Интерактивный менеджер календаря**: Полный CRUD-интерфейс на панели управления для поиска, добавления и удаления именинников из базы данных.
- 📂 **Надежное локальное хранение**: База данных дней рождения и история опубликованных постов сохраняются непосредственно внутри контейнера.

## 🛠️ Настройка и переменные окружения

Вы можете настроить ключи авторизации напрямую через вкладку **Конфигурация .env** на панели управления или задав переменные окружения:

- `TG_BOT_TOKEN`: API-токен вашего Telegram-бота (полученный у `@BotFather`).
- `GEMINI_API_KEY`: Ваш API-ключ Google AI Studio Gemini (рекомендуется для высокоточного перевода и адаптации текста).
- `LLM_API_KEY`: Ключ API для Cerebras или другого стороннего LLM-провайдера (опционально).
- `CHECK_INTERVAL`: Задержка в секундах между автоматическими проверками лент (по умолчанию `600` секунд / 10 минут).
- `MAX_AGE_HOURS`: Максимальный возраст статей для публикации в часах (по умолчанию `48` часов).

## 🖥️ Команды разработки и сборки

- **Запуск сервера разработки**: `npm run dev` (запускает full-stack приложение на Порту 3000)
- **Компиляция рабочей версии**: `npm run build` (собирает фронтенд в папку `dist/` и компилирует сервер)
- **Запуск рабочей версии**: `npm run start` (запускает скомпилированный сервер)
